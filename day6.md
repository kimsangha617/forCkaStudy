## 62. Node Affinity

Node에 Label을 적용하는 명령어

`kubectl label node node01 color=blue`

Node에 Taints가 있는지 확인하는 명령어

`kubectl describe node node01 | grep Taints`

vi 환경에서 한번에 여러 줄을 shift하기 위해서 'V + 여러줄 마우스로 드래그'하면 된다.
